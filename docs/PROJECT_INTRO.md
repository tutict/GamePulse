# GamePulse 项目简介

GamePulse 帮助开发者把分散的玩家评论整理为可搜索、可引用、可交换的本地项目。

Windows 客户端负责网页采集、文件导入、本地检索和模型问答；Android 客户端负责项目查看、文件导入、本地搜索、远程问答、证据审阅和重新导出。两端独立保存数据，不需要常驻服务器。

核心数据层使用 SQLite 与 FTS5。RAG 在本地完成规范化、查询构造、候选检索、证据重排和上下文裁剪，再把有限证据发送到用户选择的模型端点。

`.gamepulse` 项目包用于跨设备交换，包含版本化清单、SHA-256 文件哈希、项目 JSON，以及评论、标签和报告 NDJSON。项目包未加密，并明确包含评论原文。
